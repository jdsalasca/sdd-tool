# Release v0.1.29

## Highlights
- Add stricter digital-review scoring with configurable minimum score (`SDD_DIGITAL_REVIEW_MIN_SCORE`, default `85`).
- Add stronger product-readiness checks in digital reviewers:
  - architecture documentation required.
  - execution/runbook documentation required.
  - project license presence required.
- Add machine-readable delivery review report at `generated-app/deploy/digital-review-report.json`.
- Integrate reviewer scoring summary directly in autopilot logs and iterative repair flow.
