# Release v0.1.31

Date: 2026-02-08

## Highlights
- Add global `--iterations <1..10>` to control repeated human-review delivery rounds.
- Add strict validation for invalid iterations values with `SDD-1005`.
- Extend `hello` to run multi-round loop:
  - digital review
  - prioritized user stories
  - Gemini implementation
  - lifecycle quality revalidation
  - publish/update on GitHub
- Persist round history at `deploy/digital-review-rounds.json`.

## Validation
- `npm run build`
- `npm test` (80 passing)
- `npm run check:docs`
- `npm run verify:publish`

## Real Gemini Evidence
- Command: `sdd-cli --provider gemini --iterations 2 --non-interactive hello "create a lightweight notes app for teams"`
- Generated app includes:
  - `deploy/digital-review-rounds.json`
  - `deploy/digital-review-report.json`
  - `deploy/digital-review-user-stories.json`
- Round data captured prioritized stories (`P0`/`P1`) from multi-role reviewers and fed back into implementation.
