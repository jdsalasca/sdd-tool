# Strategy Docs

This folder groups long-horizon strategy and market material for `sdd-cli`.

## Contents

- `MISSION.md`: product mission and operating principles.
- `VISION.md`: long-term direction and quality bar.
- `ADOPTION_ROADMAP_90D.md`: 90-day adoption execution plan.
- `VALUE_BACKLOG.md`: prioritized improvements by impact.
- `ROADMAP_CHECKLIST.md`: milestone tracking checklist.
- `IMPROVEMENT_TODO.md`: tactical improvements and follow-ups.
- `MARKET_POSITIONING.md`: ICP, messaging, differentiation, and GTM posture.
