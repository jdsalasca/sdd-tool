# Gate to prompt matrix

This matrix ensures every gate is supported by explicit prompts.

| Gate | Required prompts |
|---|---|
| Acceptance criteria defined | "What are the acceptance criteria?" |
| NFRs defined | "Any performance, security, availability constraints?" |
| Scope defined | "What is in scope and out of scope?" |
| Test plan exists | "What are the critical paths and edge cases?" |
| Quality thresholds set | "What coverage/complexity thresholds apply?" |
| Compliance constraints set | "Any compliance or regulatory constraints?" |
| Repro steps for bugs | "Steps to reproduce?" |
| Comment audit for PR | "Which comments are valid vs debatable?" |
