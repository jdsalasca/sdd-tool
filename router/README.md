# Router scripts

These scripts define intent detection and the step-by-step conversation for each flow.

Each script includes:
- Entry signals
- Required questions
- Required outputs
- Gate checks
- Suggested agent roles
- Scripted Q/A tree with optional branches

Available flows:
- BUG_FIX
- LEARN
- SOFTWARE_FEATURE
- DATA_SCIENCE
- DESIGN
- HUMANITIES
- BUSINESS
- LEGAL
- PR_REVIEW
- GENERIC
