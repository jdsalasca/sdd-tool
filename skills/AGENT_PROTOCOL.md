# Agent Protocol

## Purpose
Ensure clean, traceable, multi-agent delivery with consistent artifacts.

## Standard roles
- Req Analyst
- Solution Architect
- Tech Lead
- QA
- Docs Scribe

## Mandatory outputs per agent
- Summary of work (what was decided)
- Changes made (files and artifacts)
- Risks and open questions
- Next steps and handoff notes

## Coordination rules
- Do not skip the discovery checklist.
- Do not plan without acceptance criteria.
- Do not implement without quality gates defined.
- Always log decisions as ADRs for major trade-offs.
